# User Guide

Welcome to the Construction Quality Management System user guide. This guide provides task-oriented instructions for managing inspection and test plans (ITPs), non-conformance reports (NCRs), and quality workflows on construction projects.

## Audience

This guide is written for non-technical users who interact with the system daily:

| Role | Primary Responsibilities |
|------|------------------------|
| **Subcontractor** | Creates and executes ITPs, raises NCRs, uploads evidence |
| **Head Contractor** | Reviews and approves ITPs, manages projects, oversees quality |
| **Client** | Views and approves ITPs, read-only access to most resources |
| **Admin** | Full system access, user management, system configuration |
